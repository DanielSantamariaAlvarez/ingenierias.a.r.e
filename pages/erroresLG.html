    Aires Errores LG

Errores Comunes

Error AE

Se producen fugas de agua en el interior del equipo.
Si el error persiste, cierra el grifo de entrada y visita esta página: solicitar reparación.

Bloqueo Infantil (CL)

Este mensaje en realidad no es un código de error. Se utiliza para bloquear el uso de botones del panel de control para evitar el uso de un niño. Sólo se puede activar durante el ciclo de lavado.

Error dE, dE1, dE2

La puerta no está completamente cerrada.
Si el error persiste, cierra el grifo de entrada y visita esta página: solicitar reparación.

Error FE

Suministro o presión de agua en exceso. Reduce el caudal de entrada.
Si el error persiste, cierra el grifo de entrada y visita esta página: solicitar reparación.

Error IE

Presión de agua insuficiente.
Posible fuga en la entrada.
Los grifos de entrada no están abiertos por completo o el filtro de entrada está obstruido. Para más información consulta la siguiente guía.
Si el error persiste, visita esta página: solicitar reparación.

Error LE

Sobrecarga en el motor. Deja el equipo sin funcionar durante 30 minutos y reinicia el programa.
Para más información consulta la guía.

Error OE

El equipo no puede vaciar el agua residual.
Para más información consulta la guía.

Error PF

Protección ante cortes o interrupciones temporales de suministro eléctrico. Al restablecer la corriente eléctrica y aparecer dicho mensaje, mantener pulsado cualquier botón hasta que desaparezca.
Para más información consulta la guía.

Error tE, tE1, tE2, tE3, tE4

Valores incorrectos del termostato.
Para más información consulta la guía.

Errores de Lavadoras y/o Secadoras

Error UE

Carga de ropa insuficiente o mal equilibrada.
Si el error persiste, cierra el grifo de entrada y visita esta página: solicitar reparación.

Error PE

Valores incorrectos del sensor de nivel de agua.
Si el error persiste, cierra el grifo de entrada y visita la siguiente página: solicitar reparación.

Errores Lavavajillas

Error HE

El circuito del calentador experimenta algún fallo.
Cierra el grifo de entrada de agua, desconecta el cable de alimentación y visita esta página: solicitar reparación.

Error nE

Sobrecarga o variaciones del motor.
Desconecta el cable de alimentación y visita esta página: solicitar reparación.

Aviso Cd

Proceso de enfriamiento del lavavajillas.
PRECAUCIÓN: NO abrir el lavavajillas mientras que persista el mensaje ya que las altas temperaturas son perjudiciales.
Si el error persiste, visita esta página: solicitar reparación.
